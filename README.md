# `YG插件排错`
1、提示插件损坏！
原因：tt.bak模块 被杀毒隔离了需要找回添加信任即可！（注win10 自带杀毒，及360杀毒卫士为主）
tt.bak模块的路径见下图我：（我的电脑输入%appdata% 回车找到YG文件夹查看）

2、yg插件工具条不见了

原因：YG插件工具条没有显示，或者安装没有成功


1）、查看cdr安装路径下draw/plugins文件夹下是否存在YG.cpg,如果不存在重新安装.


2）、cdr工具条上右键勾选YG插件工具，即可显示工具条.
